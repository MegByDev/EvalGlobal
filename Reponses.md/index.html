Terminal

  • Lister tous les éléments d'un dossier: ls
  • Créer un fichier: touch
  • Voir les fichiers cachées: ls -a
  • Supprimer un fichier: rm -rf nom du fichier
  • Commande qui permet de "nettoyer" le terminal: ~clear

Git/ Github

  • Origine du dossier:
  • Lister les différentes du projet: git branch -a
  • Initialisation de son repo git: git init
  • Envoyer modification sur github: git push
  • Changer de branch: git checkout nom_de_ma_branch
  • Commande de "capture du code" et de version: git commit
